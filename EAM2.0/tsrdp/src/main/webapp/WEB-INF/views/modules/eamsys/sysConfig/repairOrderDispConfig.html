*请选择以下一种派单方式


手动派单
手动派单是指在报修派单环节，由派单用户手动选择接单人员进行工单派单；接单人员：是指可供选择的人员，在本系统中为报修设备所归属专业的人员。


自动派单
自动派单是指在报修派单环节，由本系统根据已经设定的规则，将报修的工单自动推送到具体的运维人员待办任务清单里。自动派单的具体规则：通过专业、责任区域、排班表3要素进行派单。
步骤一，根据报修设备，判断设备归属专业，找到所有该专业的运维人员。（人员管理功能里专业为必填）
步骤二，根据报修设备位置，匹配该专业下具有此责任区域的运维人员。（人员管理功能里责任区域为必填）。
步骤三，根据排班表，匹配出当前在班人员。
步骤四，如果在前三个判断条件中，未找到相关的派单对象，则派给相关专业（报修设备所归属的专业）运维班长和主管，由班长或主管进行转单操作。
班长或主管进行转单操作：是指派给运维班长和主管的单子，由班长或主管进行接单人员（可以看到全部运维人员）的选择，从而完成转单操作。
备注：自动派单为即时（30秒之内）派单。 详细规则 收起>


自动派单+
      抢单
派单+抢单是指报修以后的工单，根据设备归属的专业，匹配出所有该专业的运维人员，该专业的所有人员都可以看到该单据，且可以进行抢单。
运维人员抢到的单子只能做接单操作，不能做转单操作。且最多只能接单， 除非已完成或已挂起，否则在抢单模式下不可再接单。在一定时间内（默认秒） 没有人接的单据，自动派给这个专业的运维主管和运维班长，由运维主管或运维班长进行转单。运维主管或运维班长只能派单，不能拒绝，对于运维主管和运维班长派下来的单子，运维人员（技师）只能接单，不能转单。 详细规则 收起>
